Synopsis:


      The project entitled as “CHAT WEB APPLICATION” has been developed as a part of a chat website using HTML,CSS,Javascript as front end,Django framework and SQLite as backend tool during MCA 1st Year.The main objective of this project is to provide a chat application using website,where the users can chat with each other.In current availability of chat applications,the user should have the app installed in the device to access and use the application to chat.But this application overcome installation procedure.

Project Description:


      Main aim of the project is to provide an online chat web platform to the users and increase the scope of chat application.This chat application is developed using Django.The chat application is person-to-person message application, which uses WebSockets for notifications.The web application can display messages which the user switches between the chats.First the user need to create an account, and other user should also have account.Then the first user could be able to search the other user.By sending request to other user,the first user would be able to chat. This web application contains text, voice messages and sending images or video.This web application is developed in Django Framework,HTML, CSS, JavaScript.Chat Application contains all the features of any chat app such as login, register and allows users to chat with anyone registered to the system.
