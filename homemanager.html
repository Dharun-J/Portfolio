Synopsis:


      The application entitled "HOME MANAGER APP" developed using Java,XML,Firebase stores the user provided information done in B.Sc CSD 3rd Year.The modules present are Reminder,Digital Locker and News. The first module contains the details that are to be submitted by the user to remind the user about the task. There are some important images or documents that are better to be stored in a soft copy manner. The second module helps the user to store the images or documents safely. Some users may don’t have time or an idea to read daily newspapers and articles.The third module helps the user to read current news and search for news according to their wish.The user does not need to search for any newspapers or carry it.They can use this module to get knowledge about the day to day life.

Project Desciption:


      The Home manager mobile application is implemented in Java programming language.The application stores information about reminding tasks,images and documents.The user has to create an account in this application. Once after registering,the user should be able to login using email and password provided.After login, the user could be able to use the features present in the application.The features are reminder,digital lock,news.Admin is able to view the customer details who all have registered and also view submitted details for the features.The user can give details to the reminder task, it is stored and the task to be done is reminded.The user can save important images and documents.They can also see daily news.The Api is called and the data is updated day to day accordingly.NewsApi is an Api that is used for obtaining news.
