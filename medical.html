       The project entitled “MEDICAL LAB SYSTEM” is developed using HTML,CSS,Javascript as front end,Django framework and SQLite as backend tool during MCA 2nd Year.The main objective of this project is to provide a website,where the patients can book lab according to their convenience timing displayed.In current availability of applications,the patient will not have access to book or view the reports.In available applications only lab incharge and doctors only will be able to access.There would be specific application or software only for the reach of the doctors lab technician.The application overcomes the problem where the patient can have the access to the system.

Project Desciption:


      The “MEDICAL LAB SYSTEM” deals with the booking for a test and obtaining reports online. The patient who is using our system for the first time can register with the details like user id, user name, password, address, mobile number and email address. Once registered they can login with their login credentials sent through email for future use. After login the patient is able to book appointments i.e., selecting the lab suggested by doctor procedure for taking the test and the patient can add a test by selecting the particular date and time. The doctor can also register with the details like user id, user name, password, address, mobile number and email address. Once registered they can login with their login credentials provided through their mail. The doctor is able to view the patient's and send reports via their login.
